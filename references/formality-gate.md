## 3. 자동추론 신호 grep

| LV | 키워드 |
|---|---|
| LV4 | 투자자·VC·심사역·라운드·시리즈·IR자료·이사회·주주총회·정부·관청·부처·공공기관·국정감사·국회·CEO·대표이사·회장·사장·M&A·DD |
| LV3 | 고객사·거래처·파트너·제안서·견적·계약서·RFP·나라장터·공공조달·입찰·외부 발신·외부 미팅·신규 고객·B2B·협력사 |
| LV2 | 사내 임원·타팀·상위리더·본부장·실장·팀장·주간보고·월간보고·분기보고·사내 공식·전사 공유 |
| LV1 | 내부·메모·복기·정리·기록·체크리스트·동료·팀원·우리 팀·내 메모·초안·드래프트·아이디어·내가 볼·나만 볼 |

### 결정 트리
- LV4 hit ≥1 → LV4
- LV3 hit ≥1 (LV4=0) → LV3
- LV2 hit ≥1 (LV3·LV4=0) → LV2
- LV1 hit ≥1 (다른=0) → LV1
- 모든 hit=0 또는 다중 LV hit → **디폴트 LV3 + 1회 핑**

### 1회 핑 표준 문구
```
🤔 수신자 확인 부탁드립니다.
이 문서, 누구에게 보내시는 건가요?
[A] 투자자·이사회·VC·정부 [B] 거래처·외부 파트너 [C] 사내 임원·타팀 [D] 동료·내부 메모
```

## 4. 입니다체 변환 매트릭스 (LV2~4 강제)

### 4-1. 동사형
| 이다체 BAN | 입니다체 |
|---|---|
| ~한다·~된다·~이다·~있다·~없다 | ~합니다·~됩니다·~입니다·~있습니다·~없습니다 |
| ~간다·~본다·~만든다·~준다 | ~갑니다·~봅니다·~만듭니다·~줍니다 |

### 4-2. 의문·과거·미래
| 이다체 BAN | 입니다체 |
|---|---|
| ~한가·~인가·~한다고·~했다 | ~합니까·~입니까·~한다고 합니다·~했습니다 |
| ~할 것이다·~한 듯하다 | ~할 것입니다·~한 듯합니다 |

### 4-3. 우회 종결 BAN (단정형 강제)
| BAN | 단정형 |
|---|---|
| ~할 수 있다·~될 수 있다 | ~합니다·~가능합니다·~됩니다 |
| ~인 것으로 보인다·~할 것으로 예상된다 | ~입니다·~예상됩니다 |
| ~라고 할 수 있다·~라고 볼 수 있다 | ~입니다 |

## 5. 면제 영역 (입니다체 강제 ✗)

- **압축미 우선:** Headline·압정 주장·HERO 첫 줄 (단정/명사형 허용)
- **단어 단위:** 표 셀·KPI·태그·고유명사·법조문·약어
- **원문 보존:** 인용박스·고객 증언·FGI 직인용·법조문 인용
- **관계격식 LV1:** 내부 메모·동료 (해라체 허용)

## 6. 안티패턴

| 안티패턴 | LV 위반 |
|---|---|
| LV4 문서에 친근체 (~예요·~네요) | 외부 결례 |
| LV4 문서에 반말 (~다·~지) | 외부 결례 |
| LV3 문서에 농담·환기 | 외부 부적절 |
| LV1 문서에 합쇼체 도배 | 과격식·어색 |
| 한 문서 LV 혼재 | 일관성 붕괴 |
| LV 결정 없이 §B 진입 | 사전생성 본질 위반 = FAIL |
| 한 문단 합쇼+해라 혼재 | AI 패턴 (R5 식별 시그널) |

## 7. 작성 토큰 단위 자기검열

### 7-1. §B 진입 직전 (룰7 메타)
> "수신자 LV 결정됐나? 어떤 LV인가? 종결·어조·호칭 LV 매트릭스 적용했나?"

### 7-2. 매 문장 작성 직전 (룰6 메타)
> "이 종결, 입니다체인가? 면제 영역인가? LV에 맞나?"

YES → 진행. 1개라도 NO → 즉시 재생성.

## 8. 우선순위

INV 22 (격식 통합) > §B 본문 산문 다른 룰. 다른 규칙(예: CEW 한 줄)이 격식과 충돌하면 격식 우선.

예: "X는 Y다" (이다체 단정) → 본문 산문 LV3이면 "X는 Y입니다"로 변환. 단 Headline/압정이면 면제.

## 9. Gotchas

| 함정 | 대응 |
|---|---|
| 본문 산문에 이다체 자연스럽게 끼움 | 사고 자체가 격식 혼재. 전수 변환 |
| Headline까지 입니다체 강제 | 압축미 손상. Headline은 면제 |
| 인용박스 원문 입니다체 변환 | 원문 보존 위반. 인용박스 면제 |
| 한 문단 합쇼+해라 혼재 | 전수 통일 |
| 우회 종결 (~할 수 있다) 입니다체 통과 | 단정형 변환 |
| 관계격식 LV1도 강제 | LV1은 해라체 허용 |
| 신호 0/다중 hit 시 임의 결정 | 디폴트 LV3 + 1회 핑 강제 |
| 1회 핑 회피 | 모호 = 핑 강제. 회피 = FAIL |
